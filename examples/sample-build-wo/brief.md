# Brief

## Problem
Landing page conversion is 2.1%, below the 3.5% target. The hero section
lacks a clear value proposition and social proof is buried below the fold.

## Scope
- Redesign hero section with clear value prop and primary CTA
- Add social proof block (logos, testimonials, metrics) above the fold
- Optimize mobile layout for thumb-zone CTA placement
- Preserve existing URL structure and SEO metadata

## Out of Scope
- Pricing page changes
- Backend API changes
- New analytics instrumentation (use existing events)

## Success Criteria
- Signup conversion rate >= 3.5% within 7 days of launch
- Mobile bounce rate does not increase
- Page load time stays under 2 seconds
