# Spec

## Hero Section
- Headline: clear value proposition (max 8 words)
- Subheadline: one sentence expanding the value prop
- Primary CTA button: "Start Free Trial" — prominent, above fold
- Background: gradient, no heavy images (performance budget)

## Social Proof Block
- Logo grid: 6 customer logos, grayscale, responsive row
- Testimonial: single rotating quote with name and title
- Metrics bar: "10k+ users", "99.9% uptime", "4.8/5 rating"

## Mobile Layout
- Stack vertically: headline → CTA → social proof
- CTA in thumb zone (bottom 40% of viewport)
- Testimonial collapses to swipeable cards

## Performance Budget
- Total page weight: < 500KB
- LCP: < 1.5s
- CLS: < 0.05
